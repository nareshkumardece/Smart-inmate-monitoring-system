# 🔒 Smart Inmate Monitoring System

## 📌 Overview
This project focuses on developing a real-time inmate monitoring system to enhance prison security using embedded technologies and wireless communication.

## ❗ Problem
Traditional prison monitoring systems lack real-time tracking and automated alert mechanisms, making it difficult to monitor inmate movement efficiently.

## 💡 Solution
Designed and implemented a system using:
- BLE (Bluetooth Low Energy) for short-range tracking  
- ESP-NOW for communication between devices  
- GPS for long-range location tracking  

The system detects when an inmate crosses a predefined boundary and triggers alerts.

## 🚀 Features
- Real-time inmate tracking  
- Boundary-based alert system  
- Wireless communication using ESP-NOW  
- GPS-based location monitoring  

## 🛠️ Technologies Used
- ESP32  
- BLE (Bluetooth Low Energy)  
- ESP-NOW  
- GPS Module  
- Embedded Systems  

## 📊 Result
Successfully developed a system capable of tracking inmate movement and generating alerts when boundaries are crossed, improving monitoring efficiency.
